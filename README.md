## About The Project

The functionality is to integrate into a single platform the documents known as checklists, which allow for work standardization and ensuring fewer deviations in the final results. The platform aims to make working with checklists more user-friendly and practical. Keeping the documentation on the platform will enable standardization and ensure that everyone is performing their work using the most up-to-date version of the documentation, grouping them by servers, sites, and clients. Additionally, it will allow attaching screenshots as evidence of the most important points in the checklists.

## Getting Started

<!-- ----------------------------------------------------------------------- -->
<!-- 3.1) Prerequisites -->

### Prerequisites

`Node.js`, `MongoDB` and `npm` are required for the setup of this web application.

Node.js: Ensure that Node.js is installed on your system. Node.js is a JavaScript runtime that enables you to run JavaScript on the server side. You can download Node.js from [the official website](https://nodejs.org/es).

MongoDB: MongoDB is a NoSQL database used to store and manage data for your application. Download and install MongoDB from [the official website](https://www.mongodb.com/es), and make sure the MongoDB server is up and running.

npm (Node Package Manager): npm is a package manager for Node.js, and it comes bundled with Node.js installation. It's used to manage project dependencies and run scripts.

<p align="right">(<a href="#back-to-top">back to top</a>)</p>

<!-- ----------------------------------------------------------------------- -->
<!-- 3.2) Installation -->
<!-- ----------------------------------------------------------------------- -->

### Installation

To get a copy of this project and run it in your local environment, follow the steps listed below.

1. Clone the repo

   ```sh
   git clone https://github.com/suazojaime/checklistlistapp.git
   ```

2. Go into the repository

   ```sh
   cd checklistlistapp
   ```

3. Install required Libraries and Frameworks

   ```sh
   npm install
   ```

4. Run server

   ```sh
   cd checklistlistapp/server
   nodemon server.js
   ```

5. Run application

   ```sh
   cd checklistlistapp/client
   npm run start
   ```
